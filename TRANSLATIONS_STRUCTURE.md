## Çeviri Anahtarları (Keys)

Tüm çeviriler şu formatta saklanacak:
```
key: "section.itemName"
language: "tr" | "en"
value: "Çevrilmiş metin"
```

Örnek:
- navbar.home (TR): "Ana Sayfa" → (EN): "Home"
- products.specifications (TR): "Teknik Özellikler" → (EN): "Specifications"
